# Student Management System

A Django web application for managing student records.

## Features

- User Registration
- Login / Logout Authentication
- User-specific student data
- Add Students
- Edit Students
- Delete Students
- Profile Dashboard
- Bootstrap UI

## Technologies Used

- Python
- Django
- HTML
- Bootstrap
- SQLite
- Render Deployment
